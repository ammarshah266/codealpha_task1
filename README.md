This is a responsive Image Gallery built using HTML, CSS, and JavaScript. The project displays a collection of images in a clean and modern layout. Users can navigate through images using Previous and Next buttons. The gallery also includes hover effects, smooth transitions, and a responsive design that works well on different screen sizes, including mobile devices.

Features:
Responsive image gallery layout
Previous and Next image navigation
Hover effects and smooth animations
Mobile-friendly design
Built with HTML, CSS, and JavaScript
